# B803：EconML：因果机器学习的实现流程

本文介绍用于因果推断的 Python 库 EconML。相比于传统在 Stata 环境中实现的 DML 方法，EconML 具备更高的灵活性和扩展性，成为双重机器学习研究的理想工具。

**突出展示的进阶应用**：

- 元学习器（Meta-Learners）的应用
- 因果森林（Causal Forests）方法
- 深度工具变量（Deep IV）技术
